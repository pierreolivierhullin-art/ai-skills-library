## Phase 0 â€” Preparation (4-8 Semaines Avant)

### Savoir Quand Lever

**Lever quand vous n'en avez pas besoin** : le meilleur moment pour lever est quand vous avez 6+ mois de runway et une traction solide. La levee sous pression de cash detruit les conditions de negociation.

**Signaux que vous etes prets a lever** :
- MRR qui croit > 10% / mois de facon consistante (3+ mois)
- 10+ clients payants (Serie A) ou LOI credibles (Seed)
- Equipe complete sur les competences critiques
- Narrative claire et simple (vous pouvez l'expliquer en 90 secondes)

## Phase 3 â€” Due Diligence (4-8 Semaines)

### Ce que les VCs Verifient

**Commercial DD** :
- Appels de reference clients (2-5 clients) : les VCs appellent directement vos clients
- Verification de la retention (churn reel vs affiche)
- Validation du pipeline (les deals en cours sont-ils reels ?)

**Technique DD** :
- Architecture review (externe ou interne selon le fonds)
- Securite basique (vulnerabilites connues, protection des donnees)
- Scalabilite de l'infra

**Equipe DD** :
- Reference calls fondateurs (ex-collegues, managers, clients)
- Verification LinkedIn (pas de faux background)
- Parfois : test psychologique ou entretien comportemental

**Legal DD** :
- Propriete intellectuelle (qui possede le code ?)
- Contrats employes et freelances (BSPCE, NDA, non-competition)
- Litiges en cours ou potentiels

### Preparer les Reference Calls

Vous pouvez (et devez) preparer vos references :

1. Prevenir les personnes citees avant qu'elles soient contactees
2. Rappeler le contexte : ce que vous levez, pourquoi vous les citez
3. Ne pas coacher sur ce qu'ils doivent dire â€” mais s'assurer qu'ils sont positifs et specifiques

## Phase 4 â€” Closing (2-4 Semaines)

### Negociation du Term Sheet

**Principe** : negocier les termes importants (valorisation, preference, board), pas les details (formatting des documents).

**Astuce** : avoir un avocat M&A startup pour la negociation finale. Les honoraires (3-8K EUR) sont largement rentabilises par les termes obtenus.

**Compromis typiques** :
- Valorisation : les VCs font souvent une premiere offre basse. Contre-proposition a + 20-30%. Atterrissage au milieu.
- Option pool : negocier la taille (10% vs 15%) et le timing (pre vs post-money)
- Preference : 1x non-participating = standard europeens. Ne pas ceder au-dela.

### Documents Finaux

**Pacte d'Actionnaires (SHA â€” Shareholders Agreement)** :
Le document juridique principal. Reprend tous les termes du term sheet en version juridiquement contraignante.

**SPA (Share Purchase Agreement)** :
La vente proprement dite des nouvelles actions.

**Statuts modifies** :
Mise a jour des statuts pour reflechir la nouvelle structure de capital et les droits speciaux.

**Closing** :
- Signature simultanÃ©e de tous les documents (usually en DocuSign)
- Transfert des fonds (virement bancaire)
- Mise a jour du registre des actions
